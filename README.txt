1、项目概述

游戏名称：卡牌匹配
目标平台：iOS
2、技术栈

引擎：cocos2d-x-3.17
语言：C++
开发工具：Xcode 14.3.1
3、项目结构

Classes/
|-configs/ 	#配置文件加载
|-models/ 	#数据模型
|-controllers 	#游戏逻辑控制
|-managers/ 	#功能管理器
|-services/ 	#业务服务
|-utils/ 	#工具类
|-views/ 	#UI视图
4、核心模块说明

由于本游戏相对比较简单，目前主要要一下几个模块

模块		｜职责
——————————————————————————
GameController	｜游戏主逻辑
——————————————————————————
CardModel	｜牌的数据
——————————————————————————
CardView	｜牌的显示
——————————————————————————
LevelConfig	｜关卡加载
——————————————————————————
GameView 	｜页面布局

5、游戏流程

启动->初始化->开始游戏->加载关卡->卡牌就位->点击卡牌->判断条件->更新手牌->胜利/失败
6、关键规则

匹配规则：相同点数或相同花色
胜利条件：主牌区清空
回退功能：撤销上一步操作
7、配置文件说明

{
    "Playfield": [...],  // 主牌区初始牌
    "Stack": [...]       // 备用牌区初始牌
}
8、运行说明

用 Xcode 打开 proj.ios_mac/PokerGame.xcodeproj
选择模拟器，点击运行
